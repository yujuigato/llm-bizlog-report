# 業務日誌智慧萃取系統 — Pain-Point Intelligence Map

從 **17 萬筆業務日誌**中，以 LLM 四階段管道自動萃取企業痛點、需求與洞察，生成可行動的業務作戰地圖。

## 🔍 專案概述

業務人員每天寫日報，但這些客戶洞察全部散落在非結構化文字中。本系統利用 Gemini 2.5 Flash 自動將日誌轉化為結構化的 L1–L7 七層銷售情報，並以 ML 聚類識別法人類型、生成痛需熱圖。

## 📊 規模

- 178,790 家企業完整分析
- 七層情報萃取（L1 現象 → L7 解決方案線索）
- 167.6 億訂單路徑分析（Sankey / Sunburst 互動圖）

## 🛠 技術棧

Python · Gemini 2.5 Flash · Vertex AI · Scikit-learn · Plotly · SQL Server
